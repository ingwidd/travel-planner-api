Travel Planner Pro API

Welcome to the backend API. Below are the available endpoints and their usage.

Authentication & Users

POST /sync-user
Payload: { "uid": "string", "email": "string" }

Syncs Firebase authenticated users with the local Postgres database.

Trips

GET /trips?userId=xxx

Fetch all trips for a specific user.

POST /trips
Payload: { "userId", "name", "destination", "startDate", "endDate" }

Create a new trip itinerary.

PUT /trips/:id

Update trip details by ID.

DELETE /trips/:id

Remove a trip from the database.

Diary Entries

GET /diary-entries?userId=xxx OR ?tripId=xxx

Fetch diary entries filtered by user or a specific trip.

POST /diary-entries
Payload: { "tripId", "caption", "photoUrl" }

Create a new photo diary entry.

PUT /diary-entries/:id

Update caption or photo of an entry.

DELETE /diary-entries/:id

Delete a diary entry.

To-Do List

GET /todos?userId=xxx&tripId=xxx

Fetch tasks. userId is required; tripId is optional.

POST /todos
Payload: { "tripId", "task_description", "is_completed" }

Add a new task to a trip.

PUT /todos/:id

Mark task as complete or update description.

Travel Tools (External Data)

GET /locations?keyword=paris

Search for city coordinates using the Photon (OSM) API.

GET /activities?latitude=xxx&longitude=xxx

Find nearby attractions and museums using Overpass API mirrors.
